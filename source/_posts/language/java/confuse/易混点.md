### 参数赋值

能够赋值的结构：

1. 数组

示例：

```java
public void test() {
  	// 二维数组
    int[][] temp = new int[1][1];
    test(temp);
    System.out.println(temp[0][0]);
}

public void test(int[][] temp) {
    temp[0][0] = 1;
}
```
